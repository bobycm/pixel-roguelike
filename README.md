# 像素地牢 (Pixel Roguelike) 🧙‍♂️

這是一套基於純前端技術（HTML5、CSS3、Vanilla JavaScript）打造的 11x11 網格 Roguelike 地牢探索遊戲。玩家將在隨機生成的地牢中探索、收集物資、擊敗怪物，並挑戰深層的強大 BOSS！

## 🌟 遊戲特色 (Features)

*   **隨機生成地圖**：採用 BFS（廣度優先搜尋）演算法確保每一層道具連通性，防止玩家被卡死，確保破關路徑順暢。
*   **無限層數與難度延續**：遊戲沒有終點，但隨著層數推進，怪物的數值會越來越強。
*   **生存與資源管理**：
    *   **飢餓值系統**：每次移動或攻擊都會消耗飢餓度，需尋找地圖上的「食物 (🍖)」補充，歸零將開始扣血。
    *   **藥水庫存**：收集「藥水 (🧪)」並存放於背包中（預設上限 1 瓶），在危急時刻按下快捷鍵即可恢復生命。
*   **戰鬥與掉落物 (金幣💰)**：採用回合制碰撞判定戰鬥，擊敗怪物可獲得金幣獎勵。
*   **動態商店機制**：
    *   每 10 層抵達絕對安全的「商人層」。
    *   可使用金幣升級「最大生命」、「攻擊力」以及「藥水庫存上限」。
    *   採用需求浮動價格（每次購買都會使該項目漲價）。
*   **史詩 BOSS 挑戰**：
    *   每 20 層必定遭遇強大的 BOSS。
    *   BOSS 具備主動追擊 AI，走位錯誤將陷入危機。
    *   擊敗 BOSS 才能開啟前往下一層的出口，且後續層次難度將全面提升。
*   **動態網頁音效 (Web Audio API)**：
    *   無須依賴外部音效檔，完全仰賴程式碼即時合成 8-bit 電子音效（包含移動、攻擊、受傷、購買、升級等回饋）。

## 🎮 如何遊玩 (How to Play)

### 啟動遊戲
本專案為純靜態網頁，無需安裝任何伺服器環境或資料庫。
只要使用任何主流的現代瀏覽器（Chrome、Edge、Firefox、Safari），**直接雙擊開啟 `index.html` 檔案** 即可立即遊玩！

### 操作說明
| 動作 | 快捷鍵 |
| :--- | :--- |
| **移動 / 攻擊** | 鍵盤 ⬆️ ⬇️ ⬅️ ➡️ 方向鍵 或 `W` `A` `S` `D` 鍵 |
| **使用藥水** | 鍵盤 `H` 或 `1` 鍵 |
| **進入下一層** | 移動角色碰觸地圖上的「門 (🚪)」 |
| **購買升級** | 在商店層使用滑鼠點擊對應按鈕 |
| **音量控制** | 使用遊戲畫面下方的音量拉桿，可隨時調整合成音效的大小 |

## 🛠️ 技術堆疊 (Tech Stack)

*   **HTML5**：架構遊戲容器、狀態列表與選單覆蓋層。
*   **CSS3**：使用 CSS Grid 繪製地牢網格，Flexbox 處理 UI 排版，並確保遊戲視窗於各種解析度置中防裁切。
*   **Vanilla JavaScript (ES6)**：
    *   負責所有遊戲的狀態管理 (State Management) 與核心迴圈。
    *   實作地圖生成、BFS 演算法連通性驗證與實體尋路。
    *   負責 DOM 操作與動畫渲染更新。
    *   建構 `AudioContext` 節點樹，動態生成並調控主音量。
