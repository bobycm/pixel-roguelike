# 像素地牢 (Pixel Roguelike) 🧙‍♂️
https://bobycm.github.io/pixel-roguelike/
## 🎮 How to Play
### 操作說明
| 動作 | 快捷鍵 |
| :--- | :--- |
| **移動 / 攻擊** | 鍵盤 ⬆️ ⬇️ ⬅️ ➡️ 方向鍵 或 `W` `A` `S` `D` 鍵 |
| **使用藥水** | 鍵盤 `H` 或 `1` 鍵 |
| **進入下一層** | 移動角色碰觸地圖上的「門 (🚪)」 |
| **購買升級** | 在商店層使用滑鼠點擊對應按鈕 |
| **音量控制** | 使用遊戲畫面下方的音量拉桿，可隨時調整合成音效的大小 |
